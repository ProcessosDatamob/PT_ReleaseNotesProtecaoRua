# Mapa para Adicionar Locais

* **Descrição**: Exibe um mapa interativo para que o usuário selecione locais confiáveis.
* **Elementos**:
  * Mapa interativo.
  * Marcadores para definir a localização exata.
  * Indicador de quantidade de locais já cadastrados
  * Sinais de "+" e "-" para possibilitar aumentar ou diminuir o mapa
  * Raio - circulo em vermelho que possibilita edição do raio
  * Campo para definição de nome do local confiável

{% hint style="success" %}
**NOTA**: É possível cadastrar um local confiável, clicando no mapa, mas para isso é preciso clicar e segurar uns segundos para que abra a opção de cadastro.
{% endhint %}

#### Confirmação de Local Confiável

* **Descrição**: Após a seleção de um local no mapa, o endereço é exibido com a opção de confirmar ou excluir.
* **Elementos**:
  * Endereço do local selecionado.
  * Botão para excluir o local.
